# Commercial and Evaluation Boundary

The public SpecShift SPS Evaluator repository is intended for inspection, research discussion, and public technical review.

Candidate-discrepancy observations, review outputs, and evaluation materials derived from bounded review discussions may not be used to train, fine-tune, benchmark, reproduce, reverse-engineer, or derive a competing system without a separate written agreement.

Any reviewer-selected blind audit, evaluation discussion, or data-sharing process would require separate written terms covering confidentiality, data handling, permitted use, and ownership.

This repository does not disclose protected implementation details, scoring internals, weighting logic, thresholds, or proprietary evaluation methods.
