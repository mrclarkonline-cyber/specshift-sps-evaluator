## Operating Boundary

This page extracts only restrained diagnostic claims.

It does not promote physics claims, consciousness claims, production-validation claims, or commercial proof claims.

## Core Public-Safe Claim

∆∆F / κ is treated as an observable second-order trajectory diagnostic.

It tests whether structured correction in a time-indexed system survives perturbation and separates from appropriate null controls.

## Safe Claim Set

### 1. Diagnostic, not ontology

Safe claim:

> ∆∆F is an observable diagnostic of second-order trajectory structure.

Do not expand this into:

> ∆∆F proves consciousness, intent, personhood, moral status, or selfhood.

### 2. Second-order structure

Safe claim:

> First-order measures describe what a system achieves; second-order trajectory measures describe how its adjustment pattern changes over time.

Safe use:

- reward versus correction structure
- output stability versus internal organization proxy
- performance versus trajectory-level signal behavior

### 3. Perturbation-sensitive organization

Safe claim:

> The diagnostic is most meaningful when evaluated under controlled perturbation rather than nominal success alone.

Safe use:

- intervention tests
- perturbation families
- degradation regimes
- authority-removal or control-scrambling conditions

### 4. Null-referenced comparison

Safe claim:

> A diagnostic result must be compared against an appropriate null or surrogate that preserves basic marginals while disrupting temporal, directional, causal, or state-conditional structure.

Safe use:

- scrambled-control null
- randomized directional null
- matched-control statistic
- surrogate perturbation
- null silence

### 5. Falsifiable by design

Safe claim:

> The diagnostic fails if null or surrogate constructions reproduce the observed curvature distribution and temporal structure within stated tolerance.

Safe use:

- explicit falsifiers
- negative controls
- failure transparency
- pre-registered tolerance
- finite-sample reporting

### 6. No mechanism access required

Safe claim:

> The diagnostic can be applied to observable trajectory data without requiring access to model weights, hidden state, private prompts, internal mechanisms, source code, or optimizer structure.

Required caution:

> Lack of mechanism access also limits interpretation. The result is a candidate discrepancy or diagnostic signal, not a full causal explanation.

### 7. Separation of performance and structure

Safe claim:

> First-order performance can remain stable while second-order correction structure degrades.

Safe use:

- performance is not enough
- success can be replayed, forced, externally supported, or environmentally simplified
- structure can fail before outcome collapse

### 8. Human review retained

Safe claim:

> ∆∆F-style outputs should be treated as candidate signals for reviewer interpretation, not automated verdicts.

Safe use:

- reviewer-controlled testing
- human adjudication
- discrepancy memo
- bounded diagnostic output

## One-Paragraph Safe Summary

∆∆F / κ is an observable second-order trajectory diagnostic for studying whether structured correction persists under perturbation and separates from appropriate null controls. It compares curvature-like behavior in time-indexed signals against scrambled, randomized, or matched-control surrogates. The method is useful only when paired with explicit falsification conditions, negative controls, and cautious interpretation. It does not infer consciousness, intent, moral status, legal status, hidden mechanism, or production reliability by itself.

## SpecShift-Safe Bridge

SpecShift may cite ∆∆F as intellectual provenance for a broader design instinct:

> first-order success is not enough; observable trajectories can contain reviewable structure about how a system reached its outcome.

But SpecShift should not be sold as ∆∆F physics or as validated production failure prediction.

SpecShift remains:

> a buyer-controlled, observable-only trajectory review layer that reviews exported workflow traces and returns structured candidate-discrepancy memos for human adjudication.
