## Send Boundary

First-send outreach should not include:

- ZIP files
- Source code
- Protected method details
- Internal scoring logic
- Training rights
- Code transfer
- Broad exclusivity
- Free-pilot language

## Approved Core Sentence

SpecShift compares observable reconciliation workflow traces against claimed final ledger or payment states and returns structured candidate-discrepancy memos for reviewer-controlled testing.

## Current Commercial Status

SpecShift is an early-stage observable-only reliability review layer demonstrated in synthetic environments. It presents plausible buyer value, but it has not been validated as live production failure-prediction.
