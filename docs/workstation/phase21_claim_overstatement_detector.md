# Phase 21 Claim-Overstatement Detector

Status: complete  
Date: 2026-06-28

## Purpose

Phase 21 adds `claim_gauntlet`, a local scanner for risky overstatement language.

## Command

Repo-local:

`python3 tools/status/claim_gauntlet.py`

Optional wrapper:

`claim_gauntlet`

## What it flags

Examples include:

- proves
- detects
- guarantees
- validates
- consciousness
- intent
- production-ready
- truth
- prediction
- autonomous
- alert
- causal
- contradiction detection
- agreement detection

## What it suggests

Safer bounded language:

- flags
- suggests
- supports
- candidate
- review prompt
- scaffold
- human-review
- not externally validated
- reviewer-controlled assessment

## Boundary

Claim Gauntlet is a local language-risk scanner only.

It does not certify legal, compliance, scientific, factual, production, or buyer readiness.
