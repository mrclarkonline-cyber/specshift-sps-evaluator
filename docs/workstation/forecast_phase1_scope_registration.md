# Forecast Phase 1 Scope Registration

Status: complete  
Date: 2026-06-28

## Purpose

Forecast Phase 1 defines the safe operating scope for the Workstation Forecast + Negative-Space Engine.

## Allowed work

The engine may create:

- forecast candidates
- negative-space review prompts
- stale-source prompts
- missing-signal prompts
- needs-more-data suggestions
- bounded next-source suggestions
- human-review queues
- audit reports

## Forbidden claims

The engine must not claim:

- prediction
- truth resolution
- alerting
- production monitoring
- causal certainty
- autonomous action
- policy, medical, legal, or financial decisions

## Boundary

A forecast candidate means a bounded review-worthy possibility.

It is not a prediction, alert, conclusion, or instruction to act.
