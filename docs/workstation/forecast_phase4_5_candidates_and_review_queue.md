# Forecast Phases 4 and 5: Candidates and Review Queue

Status: complete  
Date: 2026-06-28

## Phase 4

Creates descriptive forecast candidates from:

- negative-space watchlist items
- pattern concentrations
- descriptive math metrics
- secondary-condition candidates
- prior bounded scenario suggestions

## Phase 5

Turns those forecast candidates into human-review queue items.

Each queue item asks whether the candidate should be:

- checked manually
- supported with one bounded source
- carried forward
- parked

## Boundary

Forecast candidates and scenario queue items are review scaffolds only.

They are not predictions, alerts, action instructions, production monitoring, causal inference, autonomous action, or truth resolution.
